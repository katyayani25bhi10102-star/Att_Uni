Project Title
University Attendance Calculator

Overview of the Project
This project is a Python program that allows students to enter details of multiple subjects such as total classes held and classes attended. It calculates attendance percentage for each subject and also computes the overall attendance. The program also displays a warning message if attendance falls below the minimum requirement of 75%.

Features
• Input multiple subjects
• Calculates attendance for each subject
• Calculates overall attendance
• User-friendly console interface
• Displays warning when attendance < 75%

Technologies / Tools Used
• Python 3

Steps to Install & Run the Project
	1.	Install Python from https://www.python.org/
	2.	Save the script in a file named: attendance_calculator.py
	3.	Run the program using:
python attendance_calculator.py

Instructions for Testing
	1.	Run the script
	2.	Enter the number of subjects
	3.	For each subject, enter:
– Subject name
– Total classes held
– Classes attended
	4.	View the report:
– Subject-wise attendance
– Overall attendance
– Warning message (if below 75%)
